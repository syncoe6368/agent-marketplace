# Agent Marketplace - Contributing Guide

Thank you for considering contributing to the Agent Marketplace! This document outlines how you can help improve this project.

## Development Setup

1. Fork the repository
2. Clone your fork: `git clone https://github.com/your-username/agent-marketplace.git`
3. Install dependencies: `npm install`
4. Copy environment variables: `cp .env.example .env.local`
5. Set up Supabase project and update `.env.local` with your credentials
6. Run development server: `npm run dev`

## Making Changes

1. Create a new branch for your feature or bugfix
2. Make your changes
3. Ensure your code follows the existing style
4. Add tests if applicable
5. Commit your changes
6. Push to your fork
7. Submit a pull request

## Code Style

- Follow the existing code style in the project
- Use TypeScript for all new code
- Ensure ESLint passes: `npm run lint`
- Ensure TypeScript checks pass: `npm run type-check`

## Reporting Issues

Please use the GitHub issue tracker to report bugs or request features.
