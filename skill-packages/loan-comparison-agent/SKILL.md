# Loan Comparison Agent

> AI-powered personal loan comparison engine for the Malaysian market.

## What It Does

Powers PinjamanCompare (loans.syncoe.com) — a bilingual BM/EN platform that matches borrowers with the best personal loan offers from Malaysian banks and licensed financial institutions.

## Features

- **Multi-Lender Comparison** — Compare rates from 10+ Malaysian lenders side-by-side
- **EMI Calculator** — Reducing balance and flat rate methods with amortization tables
- **Islamic Loans** — Toggle to filter Shariah-compliant financing options only
- **Bilingual BM/EN** — Full interface in Bahasa Malaysia and English
- **PDPA Compliant** — Built-in consent management per Malaysian data protection law
- **Lead Distribution** — Auto-submit qualified leads to partner lenders

## Quick Start

```bash
# Visit live site
open https://loans.syncoe.com

# Or run locally
git clone https://github.com/syncoe6368/pinjaman-compare
cd pinjaman-compare
npm install
npm run dev
```

## Revenue Model

Cost Per Lead (CPL): Lenders pay RM 50-80 per qualified lead.
Target: RM 1,500-5,000/month from Month 2.
