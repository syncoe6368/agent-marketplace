# Code Review Agent

> AI-powered code review focusing on security, performance, and best practices.

## Overview

Analyzes codebases for security vulnerabilities, performance bottlenecks, and adherence to best practices. Supports multiple languages including Solidity, Python, JavaScript, and Rust.

## Features

- **Security Vulnerability Detection** — Catches OWASP Top 10 + Solidity-specific bugs
- **Performance Optimization** — Identifies bottlenecks and suggests improvements
- **Code Quality Metrics** — Complexity, maintainability, test coverage analysis
- **Multi-language Support** — Solidity, Python, JS/TS, Rust, Go

## Usage

```bash
./scripts/main.sh --target ./my-contract --language solidity
```
