## [1.0.0] - 2026-04-05

### Added
- BM/EN bilingual content generation engine
- Automatic SEO keyword optimization
- Multi-platform formatting (WordPress, X, Instagram, TikTok)
- A/B headline testing with CTR prediction
- Brand voice consistency engine
- Competitor content gap analysis
- Bulk generation for content calendars
- Affiliate link insertion with anchor text optimization
- Free tier: 5 posts/month, EN only, Basic SEO
- Pro tier: $19/month, unlimited posts, BM/EN/中文, Advanced SEO

### Security
- Input sanitization on all generated content
- Rate limiting on API endpoints
- No external data transmission without explicit consent
