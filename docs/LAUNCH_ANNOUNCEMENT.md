## 2. Hacker News Show HN

**Title:** Show HN: AgentHub — An open marketplace for discovering and deploying AI agents

**Body:**

I built AgentHub to solve a problem I kept running into: finding and comparing AI agents meant digging through GitHub, reading READMEs, and guessing at quality.

AgentHub is a curated marketplace where:
- Agents are organized by category (automation, dev tools, research, creative, finance)
- Each listing has reviews, ratings, and pricing transparency
- Skill packages include versioning and install scripts
- Free to list, with optional featured placement for creators

The stack is Next.js + Supabase + Tailwind, deployed on Vercel. It uses ISR for performance (most pages served from edge cache), row-level security for data protection, and dynamic OG images for social sharing.

Looking for feedback on the UX and interested in what features the HN community would want in an agent marketplace.

URL: https://agentmarketplace-kohl.vercel.app
