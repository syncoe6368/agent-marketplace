## Description (for PH listing)

**AgentHub** is a curated marketplace for discovering, comparing, and deploying AI agents.

### Why we built this:
Finding the right AI agent means scouring GitHub, Discord servers, and Twitter threads. There's no single place to compare agents, read reviews, or get setup instructions. We fixed that.

### What you can do:
- 🔍 **Browse 22+ agents** across 6 categories (Automation, Research, Dev Tools, Support, Marketing, Finance)
- ⭐ **Read verified reviews** from real users
- 💰 **Compare pricing models** — free, freemium, paid, subscription
- 🚀 **One-click install** — get CLI commands, API docs, and setup guides
- 📦 **Skill packages** with versioning and update notifications

### For creators:
- List your agent for free
- Get verified badges for established tools
- Featured placement options for discoverability
- Built-in analytics (views, clicks, reviews)

### Tech stack:
- Next.js 15 + TypeScript
- Supabase (PostgreSQL + Auth + RLS)
- Vercel (edge deployment)
- 100% server-rendered, SEO-optimized

**Free to use. Free to list. Always.**
